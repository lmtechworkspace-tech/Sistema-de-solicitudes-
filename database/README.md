# database/

Esquema de Google Sheets de SIGSO, definido en la **Fase 1** (§6 de la
especificación). Ver [`schema.md`](schema.md) para el detalle de cada hoja,
columna y su origen en la especificación.

El esquema se crea ejecutando `instalarHojas()` desde el proyecto Apps
Script `backend/setup/` (una sola vez, sobre la planilla ya creada por el
Admin — ver checklist en
`documentacion/fases/FASE-01-modelo-datos-nucleo.md`).
